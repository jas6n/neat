# That's Pretty Neat!
Flappy Bird and Pong played by the computer

flappy_game.py - flappy bird created from scratch. Uses the neat library to create a program that learns to play the game. If you let it go long enough it will keep going forever lol

game.py - classic pong created from scratch. Uses the neat library to create a program that learns to play the game with both paddles and continue hitting the ball. This also goes on forever if you let it

config-feedforward.txt - contains the configurations for the neat algorithm
